Requirements

In order to successfully use QtiPlot, you need the following libraries:

Qwt

You must use the CVS version of the Qwt 5 library which comes with the QtiPlot archive ("3rdparty" folder). This should be statically linked to QtiPlot in order to avoid conflicts on your operating system. To do this, you must change the CONFIG line of the qwt.pro file to:

# CONFIG           += dll
CONFIG           += staticlib
muParser

The last version of the muParser (1.26) library is included in QtiPlot ("3rdparty/muParser" folder), so you don't need to download and install it.

liborigin

The last version of the liborigin allows to read OriginLab OPJ project files.

qwtplot3d

The last version of the qwtplot3d is used to create the 3D plots. The library and the include files must be available. Depending on your installation, you may have to modify the corresponding line in the file qtiplot/qtiplot.pro, for example:

unix:INCLUDEPATH  += -I ../3rdparty/qwtplot3d/include
qmake

In order to build QtiPlot you also need qmake, that is official part of Qt 3.x.
