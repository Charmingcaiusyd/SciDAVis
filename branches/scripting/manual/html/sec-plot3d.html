Customization of 3D plots

theses commands are not available through any menu nor by any keyboard shortcut. They can be accessed through the 3D toolbar.

Frame

This command can be accessed by a click on the

Draw only the three axis on the active 3D plot.

Box

This command can be accessed by a click on the

Draw the three axis on the active 3D plot, and a box around it.

No axes

This command can be accessed by a click on the

Doesn't draw the three axis nor the box on the active 3D plot.

Front Grid

This command can be accessed by a click on the

Draw a grid on the front panel of the active 3D plot. The position of this grid is the plan defined by y=ymin.

Back Grid

This command can be accessed by a click on the

Draw a grid on the back panel of the active 3D plot. The position of this grid is the plan defined by y=ymax.

Left Grid

This command can be accessed by a click on the

Draw a grid on the left panel of the active 3D plot. The position of this grid is the plan defined by x=xmin.

Right Grid

This command can be accessed by a click on the

Draw a grid on the right panel of the active 3D plot. The position of this grid is the plan defined by x=xmax.

Ceiling Grid

This command can be accessed by a click on the

Draw a grid on the top panel of the active 3D plot. The position of this grid is the plan defined by z=zmax.

Floor Grid

This command can be accessed by a click on the

Draw a grid on the floor panel of the active 3D plot. The position of this grid is the plan defined by z=zmin.

Bars Style

This command can be accessed by a click on the

If the active 3D plot is a 3D histogram, this command is used to modify the style of the bars.

Dots

This command can be accessed by a click on the

If the active 3D plot is a 3D scatter, this command is used to modify the style of the data points to dots.

Cones

This command can be accessed by a click on the

If the active 3D plot is a 3D scatter, this command is used to modify the style of the data points to cones. It it then possible to modify the drawing parameters of the cones by double clicking on the plotting area.

Cross Hairs

This command can be accessed by a click on the

If the active 3D plot is a 3D scatter, this command is used to modify the style of the data points to cross-hairs. It it then possible to modify the drawing parameters of the crosses by double clicking on the plotting area.

3D Wire Frame

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to modify the style of the surface to a simple wireframe.

3D Hidden Lines

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to modify the style of the surface to a wireframe. A computation of the hidden line is done.

3D Polygons

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to modify the style of the surface to polygons.

3D Wire Surface

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to modify the style of the surface to polygons with a mesh.

Floor Data Projection

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to add a filled area projection of the surface on the floor of the plot.

Floor Isolines

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to add an isoline.

Empty Floor

This command can be accessed by a click on the

If the active 3D plot is a 3D surface, this command is used to remove any projection from the floor.
