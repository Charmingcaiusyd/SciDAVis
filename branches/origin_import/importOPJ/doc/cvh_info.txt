cvh_header_data
  describes columns in worksheets and curves or surfaces in graphs

  version   7.5    8.0
  length   0x1E7  0x1E9


curve_type 0x00:0x01
  0x01  graph
  0x10  worksheet column

curve_used 0x01:0x02 (WRONG??)
  0x00  not used in any graph
  0x01  used in some graph

data_setid 0x04:0x06
  id of the data set with the values of this column


Matrix:
  0x2B  short, width (plus function)
  0x1E  byte,  ValueTypeSpecification
  0x1F  byte,  Significant digits, decimalplaces
  
Worksheet:
  0x04  short, dataset id with values for this column
  0x12  string (12) column name
  0x11  byte, column type (X:3, 0:Y, 5:Z, 6:Xerr, 2:Yerr, 4:Label, None)
  0x1E  byte, Value type
  0x1F  byte
  0x4A  short, width (/10)
  
Graphic:
  0x04  short, dataset id (base 1) for Y (or Z in 3D)
  0x11  byte, lineconnect
  0x12  byte, linestyle
  0x15  short, linewidth (*500)
  0x19  short, symbol size (*500)
  0x1C  byte, fillarea flag
  0x1E  byte, fillarea type
  0x23  short, object id (base 1) for X (or Y in 3D)
  0x4C  byte, curve type
  0x4D  short, object id (base 1) for X in 3D
For text plots
  0x13  short, text rotation
  0x15  short, fontsize
  0x19  byte, justify mode (2: right, 26: center, left)
  0x20  byte, font properties ( &0x20: whiteout, &0x08:  bold, &0x02: italic, &0x01: underline)
  0x37  byte, x offset (/5)
  0x38  byte, y offset (/5)
For vector plots
  0x18  byte, object id shifted for column with angle, or fixed angle
  0x19  byte (short?) object id shifted for column with magnitude, or fixed magnitude
  0x56  float, multiplier (a double better?)
  0x5E  byte, object id shifted for column with end-X values
  0x62  byte, object id shifted for column with end-Y values
  0x66  short, arrow length
  0x68  byte, arrow angle
  0x69  byte, closed arrow flag
  0x70  short, width (*500)
  0x0142 byte, position (2:midpoint, 4: head, tail)
For pie plots
  0x92  byte, properties (&0x01 percentages, &0x02 Values, &0x08 positionassociate, &0x20 clockwiserotation, &0x80 categories)
  0x93  byte, format automatic
  0x94  short, distance
  0x96  byte, viewangle
  0x98  byte(short?), thickness
  0x9A  short, rotation
  0x9E  short, displacement
  0xA0  short, radius
  0xA2  short, horizontal offset
  0xA6  int, displaced sections
  
  0xC2  4 bytes, fillarea color
  0xC6  short, fillarea pattern width (*500)
  0xCA  4 bytes, fillarea pattern color
  0xCE  byte, fillarea pattern
  0xCF  byte, fillarea pattern border style
  0xD0  short, fillarea pattern border width (*500)
  0xD2  4 bytes, fillarea pattern border color
  
  0x16A 4 bytes, linecolor
  0x17  short, symbol type
  0x12E 4 bytes, symbol fill color
  0x132 4 bytes, symbol color, vector color
  0x136 byte, symbol thickness 
  0x137 byte, point offset
  0x143 byte, connect symbols flag (&0x08)
For surface
  0x13  byte, backcolor enable (&0x08), fill (&0x10), contour (&0x40), topcontourfill (&0x02)
              topcontour (&0x04), bottomcontourfill (&0x80), bottomcontour (&0x01)
  0x17  byte, surface type
  0x1C  byte, surface grid (&0x60: X, &0x20: Y, &0x40: None, XY), sidewallenabled (&0x10)
  0x1D  4 bytes, surface front color
  0x94  short, surface line width (*500)
  0x98  4 bytes, surface line color 
  0xA4  short, bottom contour line width (*500)
  0xA8  4 bytes, bottom contour line color
  0xB4  short. top contour line width (*500)
  0xB8  4 bytes, top contour line color
  0x14C short, grid linewidth (*500)
  0x15A 4 bytes, backcolor
  0x15E 4 bytes, xsidewallcolor
  0x162 4 bytes, ysidewallcolor
For 3D mesh and contour
  0x13  byte, fill contour flag  (&0x82)
