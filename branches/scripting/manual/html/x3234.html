5.10. Define surface plot

This dialog is used when you enter the New Surface 3D Plot command. It allows to create a new function of two variables. The only coordinate system is the cartesian one: z=f(x,y).

Figure 5-27. The New Surface 3D Plot dialog box.

You can then enter the X, Y and Z scales.
