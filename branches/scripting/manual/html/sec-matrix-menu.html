The Matrix Menu

This menu is only active when a matric is selected.

Set Properties...

This command open a dialog window which is used to specify some view parameters of the matrix (cell width, format of numbers).

Set Dimensions...

This command open a dialog window which is used to specify the size of a matrix. It can also be used to specify the X and Y ranges which will be used as axis ranges for a 3D-plot of the matrix data.

Set Values...

This command open a dialog window which is used to fill in a matrix with the result of a function z=f(i,j) in which i and j stand for the row and column numbers.

Transpose

Transpose the selected matrix.

Invert

Inverse the selected matrix.

Determinant

Compute the determinant of the selected matrix.

Convert to Spreadsheet

Convert the selected matrix in a table.
