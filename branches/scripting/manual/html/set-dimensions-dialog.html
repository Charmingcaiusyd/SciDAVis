Set Dimensions...

This command is in the Matrix menu. It allows to specify the number of rows and columns of a matrix

In this window, you can also define X-values and Y-values. These X and Y ranges will only be used by the 3D-plot, they are not known if you choose to define the content of the matrix with the Set Column Values Dialog.
