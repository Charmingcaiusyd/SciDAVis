How to obtain QtiPlot

The QtiPlot home page can be found at http://soft.proindependent.com. Updates and news can be found there.

QtiPlot is distributed as a package with sources which have to be compiled. Compiled packages for most Linux distributions and for Windows can also be obtained. If you want to build QtiPlot from sources, don't forget first to download and install the Qt, QwtPlot3D, liborigin, zlib and GSL libraries.
