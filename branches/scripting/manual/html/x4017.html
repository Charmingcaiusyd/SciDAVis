Installation from binary packages

QtiPlot distributes a binary Debian package and a RPM package for RedHat, Mandrake and SuSE distributions. The RPM package was created on an Ubuntu distribution.

To install on Linux Debian: download the package file qtiplot_0.8.5_i386.deb, then login as "root" and type:

dpkg -i qtiplot_0.x.x_i386.deb

To install the RPM package: download the file qtiplot-0.8.5-1.i386.rpm, then login as "root" and type:

rpm -i qtiplot-0.x.x-1.i386.rpm
