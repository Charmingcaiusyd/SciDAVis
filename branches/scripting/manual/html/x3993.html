Requirements

In order to successfully use QtiPlot, you need the following libraries:

Qwt

You must use the modified version of Qwt 4.2.0 which comes with the QtiPlot archive ("3rdparty" folder).

muParser

The last version of the muParser (1.24) library is included in QtiPlot ("3rdparty/muParser" folder) and liborigin libraries, so you don't need to download and install them.

the QwtPlot3D library is also required, and it not included in the source package. Therefore, you need to download it and install it before any installation of QtiPlot

qmake

In order to build QtiPlot you also need qmake, that is official part of Qt 3.x.
