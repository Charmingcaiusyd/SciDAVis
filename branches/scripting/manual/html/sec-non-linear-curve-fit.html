Non Linear Curve Fit

This function can be accessed by the command Non Linear Curve Fit.... It can be found in the Analysis Menu when a table is selected.

Figure 6.3. The results of the Non Linear Curve Fit....

The results are shown in the log window, the curve is plotted in the active window, and a table is created to store the fit.
